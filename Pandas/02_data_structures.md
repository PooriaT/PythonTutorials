*Pandas* supports three main different data structures which are built on top of *Numpy* array:
- Series
  - is a 1D homogenous array
  - its size is immutable


- DataFrame
  - 2D array with heterogenous data
  - it size is mutable


- Panel
  - 3D array with heterogenous data
  - it size is mutable
